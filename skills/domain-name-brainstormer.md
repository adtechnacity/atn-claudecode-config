# Domain Name Brainstormer

Generate creative domain names and verify availability for projects, products, or personal brands.

## Capabilities

1. **Analyze project context** - Target audience and key features
2. **Generate creative names** - Relevant, memorable options
3. **Check availability** - Verify across multiple TLDs
4. **Provide alternatives** - Variations when top choices are taken
5. **Explain branding rationale** - Why names work

## Good Domain Characteristics

- Short (<15 chars), memorable, pronounceable
- Descriptive yet brandable
- No hyphens (easier to share verbally)

## TLD Guide

| TLD | Best For |
|-----|----------|
| .com | Universal, businesses |
| .io | Tech startups |
| .dev | Developer tools |
| .ai | AI/ML products |
| .app | Applications |
| .co | .com alternative |

## Output Format

```
Domain Name Suggestions

## Available (.com)
1. snippetbox.com - Clear, memorable
   Why: Directly describes product

## Available (Alternative TLDs)
2. snippet.dev - Perfect for developers
   Why: .dev signals developer tool

## Taken/Premium
- codeshare.com (Taken, est. $2,500)

## Recommendations
Top Pick: snippet.dev
- Short, memorable, .dev adds credibility

Next steps:
1. Register before it's taken
2. Want more variations?
```

## Pricing Context

- Standard: ~$10-15/year
- Premium TLDs (.io, .ai): ~$30-50/year
- Taken/Premium domains: $hundreds to $thousands

## Tips

1. Act fast - good domains get taken quickly
2. Register variations to protect brand
3. Check social media handle availability
4. Verify no trademark conflicts
5. Test pronunciation aloud
